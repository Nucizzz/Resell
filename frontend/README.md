# Resell Frontend

React + Vite single-page app to interact with the inventory backend via barcode scanning.

## Getting started

```bash
npm install
VITE_API_BASE_URL=http://localhost:8000 npm run dev
```

`VITE_API_BASE_URL` indica dove raggiungere l'API (default `/api`).

## Build

```bash
npm run build
npm run preview
```
